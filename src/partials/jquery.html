jQuery

I don't think there's really many reasonable uses for yanking jQuery into a project anymore, especially because [you can do everything it does](youmightnotneedjquery.com) just by re-learning some classic vanilla javascript moves.

Having said that, I know it helps people develop better web apps faster, and hey I'm all for that. So if you want jQuery in your project, eggs-genny has got your back.

What's Included?

It is included under app/lib/js/ and will be added to your index.html file like this:

<script type="text/javascript" src="lib/js/jquery.js"></script>

The unminified version is used in development but is minified during the build process.
