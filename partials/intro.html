What the hell is an eggs-genny?

eggs-genny is the most poorly-named Yeoman generator in the history of Yeoman generators. It was invented to be a feature-rich, user friendly starting place for all web app projects big and small. Even if you have absolutely no idea how to use the command line or Yeoman, eggs-genny along with it's handy documentation should have you setting up your projects in a breeze.

WTF kind of name is eggs-genny?

Yeah I know. Dumb name, great generator. The stupid name derives from a punny line that I wrote when I initialized the repo:

An EGGS-ellent Yeoman GENNY-erator for building modern web apps.

And it's stuck ever since!
