Bootstrap CSS

Bootstrap CSS is one of the dependency options offered by eggs-genny. Note that this is just the CSS, not the JS. There are plans to bring in the Bootstrap JS stuff, but not in the near future. If you want the JS in there really badly, open a pull request.

The JS components were left out because they were seen as not as crucial as the CSS. Also they are clunky, and when eggs-genny does support them, it will be in a modular sort of 'pick your modules' thing like they have.

What's Included

The Bootstrap CSS will live in app/lib/css/ and be brought in before your stylesheet in the index.html file:

<link rel="stylesheet" type="text/css" href="css/bootstrap.css" />

The unminified version is used in development but is minified during the build process. Uncss is run on the final concatenated CSS file as well, which strips out whatever Bootstrap is not being used in you HTML files (so no worries about bloat).
